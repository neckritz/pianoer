# Pianoer

## Overview
Pianoer is a browser-based piano that supports single-hand, dual-hand, and scale play modes. It maps keyboard layouts to piano keys, lets you shift octaves, and provides quick scale changes with animated UI feedback and toasts.

## Tech Stack
- React 19
- Vite 7
- Tone.js for audio synthesis
- Framer Motion for animations
- ESLint for linting

## Local Dev
```bash
npm install
npm run dev
```
